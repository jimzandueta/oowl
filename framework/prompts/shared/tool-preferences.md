# Tool Preferences

Customize this file with your project's preferred tools, libraries, and frameworks.

Examples of what to define:
- State management (e.g. Zustand, Redux, Pinia, Jotai)
- Data fetching (e.g. TanStack Query, SWR, Apollo)
- Testing framework and utilities (e.g. Vitest, Jest, Playwright, Testing Library)
- Styling approach (e.g. Tailwind, CSS Modules, styled-components)
- Form handling (e.g. React Hook Form, Formik, VeeValidate)
- Linting and formatting tools
- ORM or database driver preferences
- Never-use list (discouraged or banned libraries)

Keep this file version-controlled. It is loaded into every implementation agent's context.
